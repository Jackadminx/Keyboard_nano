# Keyboard nano

基于CH549的多功能小键盘

<img src="./IMG/1.jpg" height="200" width="200" align="left"/>

## 功能

硬件

- 热拔插键轴
- WS2812 RGB
- CH549主控 8051
- Type-C 接口
- 3D 打印外壳

软件

- [x] 标准键盘输入，多键无冲突
- [x] 多媒体键输入
- [x] 模拟鼠标
- [x] 模拟触摸滑动、点击
- [x] 模拟 Surface Dial 轮盘
- [x] 上位机控制
- [x] WS2812 RGB 功能
- [x] 片内 EEPROM 数据保存

## 开发平台

- Keil C51 µVision V5
- 立创EDA
- Fusion 360
